# Imrekoszo Tap

## How do I install these formulae?

`brew install imrekoszo/tap/<formula>`

Or `brew tap imrekoszo/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "imrekoszo/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
